# Hammer.js 中文文档 - Hammer.Press(options)

当指针在没有任何移动的情况下按下x毫秒时，将识别Press手势。

| 选项      | 默认值 | 描述                         |
| --------- | ------ | ---------------------------- |
| event     | press  | 事件名称。                   |
| pointers  | 1      | 需要的指针数量。             |
| threshold | 9      | 在按下时允许的最小移动距离。 |
| time      | 251    | 最小按压时间，单位为毫秒。   |

# 事件

- press
- pressup
