# Product Overview

This is "冰可乐的前端笔记" (BingKeLe's Frontend Notes) - a Chinese technical documentation website built with VitePress.

## Purpose

- Personal knowledge base and learning notes for frontend development
- Comprehensive documentation covering JavaScript, CSS, HTML, browser concepts, and development workflows
- Chinese translations and documentation for popular libraries (Next.js, GSAP, html2canvas, Hammer.js, SweetAlert)
- Collection of development tools, efficiency tips, and troubleshooting guides

## Key Content Areas

- **Frontend Fundamentals**: JavaScript, ES6, TypeScript, HTML/CSS, browser knowledge
- **Development Workflow**: Git, npm, terminal tools, CSS/Sass techniques
- **Library Documentation**: Translated docs for GSAP, html2canvas, Hammer.js, etc.
- **Efficiency Tools**: Software recommendations, online tools, browser extensions
- **Troubleshooting**: Common pitfalls and solutions for npm, PC, H5, WeChat mini-programs
- **Interview Preparation**: Frontend interview questions and answers

## Target Audience

Chinese-speaking frontend developers looking for comprehensive learning resources and practical development guidance.
