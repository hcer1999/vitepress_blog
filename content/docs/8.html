NPM 实用命令总结 ​

如果你日常工作中有使用 NPM ，我相信你会喜欢这些使用技巧的。

创建项目 ​

  • 创建项目，需要输入信息

    shell
    npm init
    1

  • 以默认信息创建项目

    shell
    npm init -y
    1

安装模块 ​

shell
npm install

# 简写
npm i
1
2
3
4

一次性安装多个模块 ​

shell
npm i gulp-pug gulp-debug gulp-sass

#如果安装的所有模块的前缀是相同的
npm i gulp{-debug,-sass,-pug}
1
2
3
4

安装模块至生产环境 ​

shell
npm i gulp --save-prod

#简写
npm i gulp -P
1
2
3
4

安装模块至开发环境 ​

shell
npm i gulp --save-dev

#简写
npm i gulp -D
1
2
3
4

安装指定版本 ​

shell
npm i vue@2.5.15
1

不写入 package.json 安装 ​

shell
npm i vue --no-save
1

获取安装包信息 ​

shell
npm view xxx

#简写
npm v xxx
1
2
3
4

查看安装包最近的版本信息 ​

shell
npm v vue version
1

获取安装包完整的版本信息列表 ​

shell
npm v vue versions
1

搜索安装包 ​

shell
npm search gulp debug

#简写
npm s gulp debug
1
2
3
4

注意：此命令本人自测在淘宝镜像下无法使用，请切换回默认镜像。

卸载包 ​

shell
npm uninstall vue

#或者
npm rm vue
npm un vue
npm r vue
1
2
3
4
5
6

这个命令会删除node_modules文件夹及package.json中对应的包。

在保留 package.json 信息 ​

shell
npm rm vue --no-save
1

依赖枚举 ​

shell
npm ls
1

此命令会将本机所有的项目安装的所有依赖列举出来

查看当前项目的依赖列表 ​

shell
npm ls --depth=0
1

查看全局安装的依赖 ​

shell
npm ls -g -depth 0
1

过期依赖枚举 ​

shell
npm outdate
1

执行脚本 ​

查看可执行的脚本 ​

shell
npm run
1

然后根据 package.json 中的配置来执行，比如：

shell
npm run serve
1

安装 Github Repo 上的包 ​

shell
npm i https://github.com/sindresorhus/gulp-debug
1

忽略域名安装 ​

shell
npm i sindresorhus/gulp-debug
1

打开安装包的 Github 主页 ​

shell
npm repo create-react-app
1

列出所有 NPM 环境的可用变量 ​

shell
npm run env | grep npm_
1
